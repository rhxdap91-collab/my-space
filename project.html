مشروع My Space هو منصة رقمية تعرض خبراتي ومهاراتي بطريقة جذابة وعصرية. يمكن للزوار استكشاف المشاريع والتقنيات بسهولة وفهم التفاصيل بشكل مباشر.

فكرة المشروع

تحويل السيرة الذاتية التقليدية إلى مساحة رقمية تفاعلية تعرض المشاريع والتقنيات المستخدمة لكل مشروع .

مميزات المشروع

  • تصميم عصري وجذاب.
  • واجهة سهلة الاستخدام والتنقل.
  • أقسام واضحة للمشاريع والمهارات.
  • تفاعلية محسّنة باستخدام JavaScript.

التقنيات المستخدمة

  • HTML لبناء الهيكل.
  • CSS لتصميم حديث وجذاب.
  • JavaScript لإضافة التفاعلية والحركات.

هدف المشروع

عرض خبراتي ومهاراتي بطريقة رقمية مبتكرة تجعل تجربة الزائر ممتعة وسهلة الفهم.
